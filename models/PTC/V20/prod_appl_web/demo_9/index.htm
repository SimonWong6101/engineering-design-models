Introduction
In this demonstration, we will be working with a solid model representation for an entire boom box assembly. We will focus on one particular part, a plastic injection molded battery housing. You will see a complete mold assembly for this part including: core, cavity, sliders, ejector pins, etc. Then you will see our process planning tools which document assembly instructions for the shop floor and tools for the manufacturing process planner. In addition, we will look at how electrodes can be created to burn the final mold geometry and we will show toolpaths to machine the mold geometry as well and the electrodes. You will see post processing to provide machine ready NC code, and programming for CMM's. Finally we will show you how to compare data from computer aided inspections machines such as CMM's and laser scanners with the design model. Then we will show you something no one else is able to do. We will make drastic modifications to the battery housing including: changing existing geometry and adding new geometry to the model. You will then see all of the deliverables be updated by the software to remain consistent with the design. When you see this change try to imagine how much time and money this would save you compared to the current process you are using at your shop.
