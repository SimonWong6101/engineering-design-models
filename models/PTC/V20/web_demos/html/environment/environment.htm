Pro/MFG Environment


INFO BOX
All information relating to the sequences and operations being generated is available on line via the information box. Where relevant it also gives us the opportunity to make in stream changes to machining parameters or tool information.

OPERATION
WORKCELL
Definition of the workcell provides the system with the information required to understand the physical capabilities of the machine. These include the basic type of machine - mill, lathe, etc. and also the fundamental parameters such as maximum feeds, speeds and travels in the available axes.
Workcell definitions will be contained in a library such that they can be accessed as required. This means that following initial definition, this information is available for all future work.

COORD SYS
Coordinate systems are used to define the origin for CL DATA and to provide orientation for the sequence. A coord. system is defined at the operation level. This can exist anywhere within the manufacturing model. For this reason it is extremely easy to define a number of operations each with their own coordinate system which will be for different fixture setups with the workpiece in different orientations. It is also possible to define a csys specifically for an individual nc sequence.

TOOL TABLE
Tools are defined and given a descriptive name, this is added to a tool table for a workcell. Within this table there is a field for tool number. This is the value that is output in the LOADTL CL command. By formating the tool table this way it is possible to ensure that the correct tool number is output for a sequence even if there is more than one tool of the same type in different positions on the machine.

LIBRARIES
It is easy to set up libraries for all the important machine and tool information. Information such as machining parameters and tool parameters can be made available from a library area. This will ensure use of standard sets of data by all users. In addition it is possible to configure a machinability database which will set feeds and speeds for milling, turning, and holemaking based on material, condition and tool dia.
