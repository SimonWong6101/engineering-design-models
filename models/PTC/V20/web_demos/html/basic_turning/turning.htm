2 Axis Turning


Pro/MFG provides the NC Programmer with a wide suite of tools for lathe programming. In the following example, a locking fitting is to be machined from slightly oversized bar stock. As you can see, the finished design will require both ID and OD turning, and the slots on the back end must be milled as a secondary operation.

To program this part, we first imbed the part inside the bar stock and define where the programming origin should be located. We don't need to worry where the part was constructed in 3-D space. The NC Programmer places machine zero. In this case we have set it on the finished part surface.
 

We can always look at our model in a shaded view or in wireframe. Our first operation is face off 1 /4" of stock from the front. We've chosen a Mazak turning center and a standard 80 degree insert. We've chosen to take off .05" with each pass, and we can view the resultant path in a number of manners. We will start with a wireframe display of the tool path.
 

  Notice on the final pass, we machined to the finish part surface. Some of the options we have at our disposal include parameters for stock allowance. Suppose we intend to come back with a second tool later, so now we want to leave .030" with this tool path. Without any changes to geometry, we simply modify the parameter in the table.

Let's move on the outside diameters. We will use the same roughing tool, and the system knows that the tool will now move in a lateral direction to work on the outside. Next, we identify what we wish to machine. We could simply pick a starting and ending surface and generate the tool path across the entire part, but we want to be a little more specific. We want the tool path to ignore the grooves for now, as we will come back with a groove insert later. Second, we don't want to machine all the way back to the far end, since the part is held in a chuck and we can only machine up to its face. Accordingly, we will define this tool path using some of the part profile and some of our own constraints.
  

Notice that we currently have the stock allowance set to .015" and that the system has offset the tool path accordingly. I can also place different stock allowances on the x and z surfaces.

We also have options for finish machining. Maybe after the roughing paths, we want a cleanup pass across the entire area, leaving .005" for the finish tool.

 
We can use Pro/NC-CHECK to display the tool path as a solid model simulation.
 

Now we're ready for our finish tool. Notice how tools are defined in Pro/MFG: we provide the system with the insert and holder dimensions such as lead and trailing angles, nose radius, tool holder width and turret station number.

For this sequence, we will be machining the same geometry as the roughing sequence, but using a different tool and making only profile passes. We can even have multiple profile passes, removing small amounts of material with each pass (or even no more material, for a spring back pass).
 

Finally, notice as we turn on the shaded mode, after each tool path is created we can also show the material that was removed, helping us to keep track of where we are at and allowing us to make drawings showing in-process part configurations.
 

Now let us program the two ring grooves. We have selected a groove tool, the groove surfaces, and our machining parameters. All of the stock allowance parameters described above are applicable here as well, along with the capability to rough or profile or both. Various types of machining motions are also available.
 

Finally, let's finish this part with the inside diameter turning. Pro/MFG knows to rotate the tool appropriately, and the same roughing and finishing sequences are available.
