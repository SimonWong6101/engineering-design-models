Manufacturing Process Information



Maunfacturing Process Information in the form of route sheets, bill of materials, and customized reports can be created for manufacturing models.

The Route Sheet contains information on the machine and processes used for each sequence including, volume of removed material, set-up time, and run time.
 


A Bill of Materials can be created for the manufacturing model including a tool list.

A Customized Report for the manufacturing model can be created using Pro/REPORT. Information in this report can include operation, workcell, and NC sequence parameters. The report is created in a drawing of the manufacturing model.
