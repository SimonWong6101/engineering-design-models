Generating NC Programs for Drilling


 
Pro/MFG provides an extended set of functionality specifically for generating programs for drilling.  The manufacturing functionality leverages the information defined in the solid model that describes hole-type features.



Pro/MFG enables the selection of holes to be drilled by picking up the hole features in the design model. Holes can be selected depending on their size, location, or by pre-defined groups. In this case we have selected six (6) holes by diameter and the surface on which they are to be drilled.


Notice how the holes are drilled in a particular order. This order is controlled by the scan type parameter or it can be set up by picking the holes in the order to be machined. We can also explicitly define the path that the drill will take between the holes by sketching the trajectory.
 

The hole depth can be controlled automatically.  In this case you can see how these holes have different starting surfaces and depths, this is all recognized  automatically. The depth can also be controlled by the drill point or shoulder. 
 


For the four (4) counter-bored holes in this component there are three different drilling cycles utilized.
1) Pre-Drill
2) Drill
3) Counter Bore
It is only necessary to identify these four holes once and then have the system create the inner dia as a through hole then drill to depth using the tool tip to calculate depth for the socket diameter and then use a facing sequence to generate the flat at depth that is required.
 


On this component there are some additional holes to be drilled through multiple webs. We can utilize the web drilling functionality to generate a drilling sequence that will drill at feed where there is material to be removed, and rapid between the web spacing.
 


Finally it is very easy to recognize the geometry of the chamfers to setup the countersinking sequence.
