Plunge control
When cutting steel molds, cavities, castings, etc., it is sometimes desirable to plunge into the material at the same location for each z depth.



Pro/MFG provides several tools to define the approach and exit position for tool paths. This functionality provides safe entry and exit conditions for the tool path. For example, an approach axis or point may be specified so that the tool will move to each z depth at that location. This prevents the tool from plunging directly into the material. The roughing tool paths then use this hole as their safe entry location to move to the next depth for each roughing slice.
