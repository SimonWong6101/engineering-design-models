Plunge Milling

Plunge Milling cuts in the direction of the Z-axis. This means material removal can be maximized because the force is along the tool axis and therefore tool deflection is minimum. Flat endmills, Radius endmills and the new plunge milling tools are supported for Plunge Milling.


Plunge Roughing a Flat Surface : 



This sequence takes a flat endmill and roughs the top part surface. Plunges are taken using a tool path which moves the tool back and forth across the entire surface.
 

Plunge Roughing a Pocket: 



This sequence takes a bull endmill and roughs a 2 level pocket using a spiral scan type.
