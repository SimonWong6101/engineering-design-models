4 Axis Rough Turning






Pro/MFG provides a full suite of tools for multi-axis turning. Pinch turning tool paths can be created quickly and easily from a single tool path. In addition, any combination of tool paths from Head 1 and Head 2 can easily be synchronized to optimize cutting time.



4 Axis Simultaneous Roughing
In the following example, a 4 axis (pinch turning) tool path rough machines the shaft from the bar stock. The user simply creates the tool path for Head 1 and Pro/MFG automatically determines the correct tool path for Head 2.



As we zoom in on this tool path, notice the correct offsets, step depths, and synchronization have automatically been applied to the resulting tool path. In addition, any of these values can simply be changed by modifying parameter values.
