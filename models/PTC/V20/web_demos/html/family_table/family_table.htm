Programming Families of Parts

Pro/Engineer family table functionality greatly reduces the programming time for families of parts. Family tables enable a user to define the varying dimensions, features, parameters, and components of a model. By entering different values into the rows and columns of this table, new variations (instances) are quickly created. Once the manufacturing process is captured for one of the instances, any of the instances of that model can easily be replaced to be the current manufacturing model. Upon regeneration, the tool paths are automatically updated according to definitions defined in the family table.

This utilization of Family Tables works in all manufacturing solutions offered by Parametric Technology Corporation including: Milling, Turning, and Wire-EDM. The following examples demonstrate the power of family tables for turning and milling applications.
