4 Axis Machining


The addition of a programmable rotary table is a very popular addition to the traditional 3 axis milling machine. Industries such as aerospace, petrochemical, and medical all require various types of radial machining. Programming 4 axis operations, while difficult or impossible using 2-D drafting based CAM systems, is very simple in Pro/MFG. Upon defining a 4 axis milling sequence, the system automatically outputs both the tool location and its orientation.



The following example is a downhole drilling bit for an oil well operation. This shows the rough and finish passes used to machine the teeth on the first row.


The next example is a medical plastic insert with a radial hole pattern. By selecting any single hole, the system can automatically create the toolpath for the entire hole pattern, including necessary rotation commands.




The final example is another downhole drilling casing. This shows a 4 axis milling of helical slots along the side of a cylindrical part.
