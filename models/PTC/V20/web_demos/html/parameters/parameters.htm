Setting and Modifying Manufacturing Parameters

There are several ways to set and modify parameters within Pro/MFG:
- by using the Parameter Tree;
- with a Machinability Database;
- directly from the Model Tree.

Parameter Tree
Machinability Database Files
Model Tree


Parameter Tree
The Parameter Tree provides easy access to parameters at the NC Sequence and Cut Motion levels. Relations may be set up within the Parameter Tree so that when a parameter value is changed, related parameters will update accordingly.

View the Parameter Tree with Relations


Machinability Database Files
Machinability database files can be used to initialize feeds and speeds according to the workpiece material and the tool to be used.

View a Machinability Database File in Pro/TABLE
 

Model Tree
Parameters can also be modified directly from the Model Tree. This saves time when parameters must be changed for several NC Sequences -- the parameter may be modified in the Model Tree for each sequence, and the manufacturing model may be regenerated after all changes have been made.  Parameters may also be defined within the definition of the sequence.
