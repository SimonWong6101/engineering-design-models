Simplification of Geometry for NC Programming

Often, complex components need to be programmed for multiple operations, setups, or machines. Traditional programming systems on force the user to display all of the geometry at all times. Pro/ENGINEER enables the users to easily simplify the model to only the required geometry for that machining step. This functionality is called a Simplified Representation. Creating the Simplified Model
Simplified representations provide an easier method of machining a part or assembly that has multiple machining operations. This example shows a part that starts as a casting.
  • The first machining steps are performed on a lathe, turning it down to net shape.
  • The turned part is then machined on a 4 or 5 axis milling center.
While programming the turning operations, it would be easier if the milled slots were not visible in the model, and this is precisely the functionality Simplified Representations provide.

A simplified representation is created to only show the geometry required for the lathe operations.

Lathe Work Representation
The Simplified Representation is then utilized in programming the required operations.


Mill Work Representation
With the lathe work complete, the full model representation is utilized in manufacturing to generate programs for the milling machine.
