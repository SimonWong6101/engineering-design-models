Generation of Cutter-Location (CL) Data Files for Post Processing


Pro/Manufacturing enables programmers to define the output of the tool paths to a Cutter-Location Data file for post processing. These are:

    Output Control - Tooling
    Output Control - Toolpath
    Output Control - Sets of Tool Paths
    Output CL-DATA/POST PROCESSING



Output Order - Tooling To minimize the number of tool changes in a job, you may order the tool path output by tool. The programmer selects each tool in the order to be machined. The tool paths that utilize the selected tools will be output together, minimizing tool changes.




Output Order - Individual Tool path
The programmer has the ability to set the order in which the tool paths are output by selecting the tool path by name. The output CL Data will be generated based upon the order in which the tool paths were selected.
 



Output - Sets of Tool Paths
If you need to output a specific group of tool paths, the programmer may do so by defining an Output Set. Simply check off ones you want to output from the master list of tool paths.
 



Output CL/POST-PROCESSING
You may output the CL Data File, access a post-processor or execute both options similtaneously. The post may be run interactively or the job may be batched in the backround or on another computer while you continue to work.
