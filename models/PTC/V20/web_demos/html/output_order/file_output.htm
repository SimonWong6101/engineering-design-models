Pro/Manufacturing Tool path Output Control



There are several methods for controlling the way tool paths are output from Pro/NC modules:
Output by Tool
Output by Toolpath
Output by Defining a Set



Output Order By Tool
To minimize the Number of tool changes in a job, you may order tool path output by tool, by selecting each tool in order you wish the tool to be loaded, all tool paths associated with that tool will be output together, minimizing tool changes.





Output by Individual Tool path
Define the order of tool paths output by picking each tool path from the list. Each tool path it is output in the exact order picked.





Output by a Set
If you need to output a specific group of tool paths, you may by defining a 'output set'. Simply check off ones you want to output from the whole list of tool paths.
