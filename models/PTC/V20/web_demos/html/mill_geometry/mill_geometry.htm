Pro/MFG Definition of Machining Geometries


Pro/MFG provides several tools to capture the manufacturing geometry to be machined while maintaining the machining intent.  For example, the user can select surfaces or edges to be machined directly fromt the solid models. In addition, the user can also define additional geometry to specify the areas to machine. In either case, the definition of the geometry may be re-used in various NC sequences within the manufacturing model.

Mill Volumes
One way to define the geometry to be machined is with MIll Volumes. This is the volume to be removed during an NC sequence. Volumes can be quickly and easily created in several ways. One way to quickly create a Mill Volume is by selecting the seed and bounding surfaces from the design model. The system will then create the volume by gathering the seed surface and all its neighboring surfaces until it reaches the bounding surfaces. The resulting surfaces will define the volume.




Selection of Surfaces from Model
Another way of defining a Mill Volume is to simply select each continuous surface from the design model. 




Mill Windows
A Mill Window can be created by sketching or selecting a contour in the retract plane. All surfaces within the contour will be machined. This window below was easily created by using the sketcher tool "USE EDGE" and selecting the "chain" of edges from the design model. By selecting edges from the design, we are guaranteeing the associative link between the design and our tool paths.


 


Mill Surfaces
Mill Surfaces are special surface features created by a variety of methods that can be used in surface milling NC sequences. These "quilts" can also be utilized in other BLANKS such as creating Mill Volumes and specifying start and end surfaces for holemaking.




Turn Profiles
Turn profiles are used for turning sequences. A Turn Profile can be defined at any time the tool path creation process and can be referenced in more than one Turning NC sequence. For instance, the same Turn Profile can be used when creating a rough, semi-finish and finish NC sequences.
