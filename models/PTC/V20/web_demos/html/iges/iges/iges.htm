Machining of IGES Data

In todays manufacturing environment, it is not uncommon for comapnies to utilize many different CAD/CAM tools. IGES data is a common translation format from a design company to a manufacturing job shop. In order to be competitive, a CAM system must be able to import and machine IGES data to address the needs of the manufacturing community.



    Importing of IGES surfaces
The first step in machining an iges file is importing the data into Pro/Engineer. Pro/Enginner not only provides excellent  import capablities, which adhere to the lastest standards, but functionality to tweak and fix  imported geometry.  Once the geometry is imported into Pro/Engineer it may then be brought into Pro/Manufacturing and toolpaths applied.
 



Rough Machine IGES Profile

The first toolpath will machine around the outside profile of  the iges model. This pass will define the outside contour of the part and also provide clearance for the the surface passes which follow. The iges part is imbedded in a solid block giving us the advantage of using NC Check with imported geometry. We pick the edges we want to machine directly off the iges model . These edges do nothave to lie in the same plane. A seperate pick for height will determine the depth of the cutting tool. The edges merely define the outside shape as if we were looking at a 2D profile.

 





Rough Machining of IGES surfaces

Our second toolpath will rough machine the entire top of the part. A bull nose end mill is used machining at different step depths. The surfaces to be machinined are selected and stock allowance added from the parameter file. The roughing passes are automatically calculated based on the size of the stock, step depth,stock allowance and the surfaces picked to be machined.

 



Finish Machining of IGES surfaces

The last toolpath finish machines the iges surfaces using a ball endmill.  Pro/Manufacturing allows you to have all the surfaces selected during the roughing sequence carried over to the finish tool. There is no need to re-select surfaces. By using a new tool and a few new parameter settings a finish toolpath can be quickly and easily generated.
