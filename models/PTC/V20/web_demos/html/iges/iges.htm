Machining of IGES Data

In todays manufacturing environment, it is not uncommon for comapnies to utilize many different CAD/CAM tools. IGES data is a common translation format from a design company to a manufacturing job shop. In order to be competitive, a CAM system must be able to import and machine IGES data to address the needs of the manufacturing community.


Importing of IGES surfaces
The first step in machining an iges file is importing the data into Pro/ENGINEER. Pro/ENGINEER not only provides excellent  import capablities, which adhere to the lastest standards, but functionality to tweak and fix  imported geometry.  Once the geometry is imported into Pro/ENGINEER, toolpaths can be applied directly on the unrepaired IGES or the user can easily repair the IGES or convert it into a fully evaluated solid model.
 




Rough Machining of IGES surfaces
The iges part is embedded in a solid block giving us the advantage of using NC Check with imported geometry. We pick the surfaces we want to machine directly off the iges model. Our first toolpath will semi-finish machine the entire top of the part. The surfaces to be machinined are selected and stock allowance added from the parameter file. The semi-finishing passes are automatically calculated based on the size of the stock, step over,stock allowance and the surfaces picked to be machined.
 


Finish Machining of IGES surfaces
The next toolpath finish machines the iges surfaces using a ball endmill.  Pro/MFG allows you to have all the surfaces selected during the roughing sequence carried over to the finish tool. There is no need to re-select surfaces. By using a new tool and a few new parameter settings a finish toolpath can be quickly and easily generated.
