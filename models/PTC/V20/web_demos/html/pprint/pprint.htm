Utilizing PPRINT Commands for Operator Messaging


PPRINT commands provide a convenient mechanism to display text information within an NC part program. Displaying tool tables, offset registers, operator messages are typical uses for the PPRINT command. Pro/MFG gives you several easy ways capture and display your important text messages.

    PPRINT SetupTable
    PPRINT CL-Data Example


PPRINT Setup Table
Pro/MFG provides a PPRINT setup table so that you may quickly define common PPRINT requirements. The programmer simply enters YES/NO for the desired information to be output in the CL Data file.
