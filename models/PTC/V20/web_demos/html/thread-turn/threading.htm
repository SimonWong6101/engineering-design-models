Thread Turning and Milling



Pro/MFG enables a programmer to easily generate tool paths to machine threads on either a lathe or milling machine. Both internal and external threads can be machined. Also, a face thread sequence (consisting of a single radial move) can be programmed for a lathe.

Thread Turning supports ISO standard thread output. The different thread types include Unified, Acme, Buttress, and User Defined.

Pro/MFG also provides a set of turning thread parameters which can be set to further define the threading sequence.
 

Thread milling enables the creation of helical milling of cylindrical surfaces. The tool path is also further controlled by setting milling thread parameters.
