Pro/MFG - Lathe Live Tooling

Many modern lathes are equipped with milling spindles which gives greater flexiblity in programming parts completely on one piece of equipment.

Pro/MFG provides powerful tools for programming lathes with live tooling. The definition of which head will be milling/drilling, orientaion, and approach/exit all are controllable within Pro/MFG.

Pro/MFG supports this equipment with up to 5-axis milling control. OD/Face slots, holes, or text can be programmed and simulated directly.

Milling Text
Often text characters such as part numbers, labels or revisions must be engraved on a part while it is on the machine. Pro/MFG provides this by choosing the whole text directly. It is cut automatically in one shot. With this powerful functionality, it is no longer necessary to program letter by letter.


 
 


Multi Axis Hole Drilling
Holes in lathe parts can provide unique challenges to programming a mill/turn center. Pro/MFG gives multi-axis tool control for programming holes on shafts or castings.
