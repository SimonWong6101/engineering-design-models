Customizing Toolpaths



Pro/MFG provides the user with interactive low level control of the tool for steering clear of fixtures and clamps. This functionality is available in a simple and powerful user interface to quickly create the desired output.

Approach and Exit Motions
Many times, special entry and exit conditions have to be met when machining a part in order to avoid fixtures or to get rid of burrs. This can be preset by specifying parameters in the parameter file or by creating the desired motions under the 'Customize' menu.







Interactive Motions
Besides Approach and Exit Motions, the programmer also has control of where the tool needs to go during the cut. Pro/MFG allows the user to GoTo a point, Go Delta from the current tool position, Go Home, Plunge, Retract, or Sketch a tool path.





Go Delta
Go Delta gives you the option to move the tool from its current position an incremental amount in any combination of X, Y or Z.




"Customize" User Interface
A new user interface has been added to easily control the customized motions described above. The Pro/MFG user has the ability to cut and paste common customized tool motions as well as copy, redefine, delete and modify parameters for each motion. When a tool motion is selected in the dialogue box, the corresponding tool path will highlight on the main viewing window. It makes it very easy to see what each tool motion is doing and in what order.
