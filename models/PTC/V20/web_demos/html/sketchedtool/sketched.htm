Creating and Using a Sketched Tool

A few applications of sketched tools include edge chamfering, corner rounding, or routing bits. Pro/MFG provides a method to sketch a tool's profile and cut geometry by driving the tool along an edge on the part. This is accomplished with the use of a control point on the tool's sketched profile.
