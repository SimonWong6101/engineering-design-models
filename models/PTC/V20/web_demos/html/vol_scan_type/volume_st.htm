VOLUMES - Ways to Cut a  Cavity

This set of toolpaths machine a simple casting cavity. It is intended to illustrate the use of different  tool control values for getting the most suitable automatic tool path and the behavior seen in relation to internal bosses upstands or islands within a pocket cavity.



The component also illustrates the use of ramp angle and helical movement between step depths for high speed material removal.

    Zig-Zag milling
    Modifying the Cut angle
    Cut in one direction only
    Spiral Cutting
    Automatic Island Avoidance
    High Speed Helical Plunge Entry
    Profile Side Surfaces
    Cut one pass at Full depth
    Multi Surface Cutting


In addition it is possible to make a change to the component to add an additional internal boss in order to show how tool paths can automatically update because they are associatively linked to the model geometry.

In this component it is required to machine to the flange depth before clearing the material from the remaining lower area of the cavity. This is achieved easily by defining each roughing region. By separating the milling pockets, it is very straight forward to let Pro/MFG-MILL know which areas of the component we are cutting.

The first machining sequence cleans down to the flange height. The shape of the part here will demand control of the way that the tool will lace across the work piece. Initially we will see the default tool path with the tool moving in straight lines.
 

To change the way a tool cuts it is only necessary to alter a couple of the toolpath settings.

For example the component is longer in one direction than the other so modifying the cut angle will give us more continuous cutting.
   

If a part requires a special finish or the machine has poor positioning you may set the toolpath to only cut feed in one direction.
 

For this type of pocket with a somewhat circular profile it is much more effective to utilize a spiral motion.

 

MAIN VOLUME
Within the main cavity are internal bosses or islands which must be avoided. Pro/MFG will develop the tool path automatically allowing for island avoidence by setting the area scan type value.

When cutting hard materials, such as tool steel, a straight plunge into the material is not desirable because of possible tool breakage or excessve tool wear.

Pro/MFG can easily control the way a tool plunges by introducing a ramp angle and/or a helix diameter which will provide a spiral downward plunge movement.

 

PROFILING
Another requirement for this type of geometry is to clean up the walls which have both draft and fillets associated with them. This is achieved using an automatic profiling sequence using an angled radiused cutter.
 

It is probably ok to run a single at depth profiling pass here, again this is achieved by changing the tool path setting. The step depth is simply set to a value greater than the profile surface height.

 

CUTLINE
To clean up the radius at the top of the side wall, we can follow the contours of the multiple surface fillet with one continuous tool path using a ball endmill.
