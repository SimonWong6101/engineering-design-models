Volume Milling Techniques



This set of sequences machines a simple cavity for a casting. It is intended to illustrate the use of different scan types for getting the most suitable automatic toolpath and the behaviour seen in relation to internal bosses upstands or islands within a volume. The component also illustrates the use of ramp angle and helical movement between slices for high speed material removal. In addition a profile sequence is included to illustrate the capability to clean up the drafted/filleted side walls. It is also possible to make a change to the component to add an additional internal boss in order to show associativity between geometry and toolpath.

    Multi Region Volume Milling


In this component it is required to machine to the flange depth before clearing the material from the remaining area of the cavity. This is achieved easily by defining these areas as individual milling areas or volumes. By doing this it is very straight forward to let Pro/MANUFACTURE know which areas of the component we are intending to deal with at any given time.

The first machining sequence cleans down to the flange height. The shape of the part here will demand control of the way that the tool will scan across the workpiece. Initially we will see the default tool path with the tool moving in straight lines.


In order to achieve something more effective it is only neccessary to alter a couple of the sequence parameters which control all aspects of machining. For example the component is longer in one direction than the other so modifying the cut angle will give us more continuous cutting.




It is also easy to control the sequence to only cut in one direction.


For this type of volume with a somewhat circular profile it is perhaps even more effective to utilise a spiral motion, again a simple modification to the parameter set achieves this. Because this motion is very effective for this kind of geometry it is possible also to select a larger tool.





MAIN VOLUME
For the main volume the geometry contains internal bosses or islands which require to be avoided. Pro/MANUFACTURE will achieve this automatically again allowing for the scan type to be easily controlled. In the first sequence the parameters are set to have the tool retract on encountering an island.


It is also very easy to control the tool to pass around the internal islands. There are a number of possibilities here, one is to have the tool follow the profile of the islands on encountering them.


You will have noticed how the tool in the previous sequence followed the profile of the islands with the tool continuing across the component. Another possibility is to have the tool change direction each time it encounters an island. Again this is entirely automatic depending on parameter selections.


Because of we are machining female geometry here it is necessary to have the tool plunge to each slice within the material to be cut. It would be possible to machine a small area initially to give access for the tool. It is also possible to modify the tool movement to reduce the effect of plunging into the material. This is done here by providing a helical plunge movement between slices. Again this is controlled very simply by modifying parameters.





PROFILING
Another requirement for this type of geometry is to clean up the walls which have both draft and fillets associated with them. This is achieved using an automatic profiling sequence using an angled radiused cutter.


It is probably ok to run a single at depth profiling pass here, again this is achived by manipulation of parameters. The step depth is simply set to a value greater than the profile surface height.



CUTLINE
It is necessary to also clean up the rad at the top of the side wall. It is possible here to ask the system to follow the contours of the fillet surfaceswith a ball nosed cutter to achieve this.



Volume Milling - Core Geometry




In order to machine this type of geometry we can utilise volume machining in a similar way to that used for cavity geometry. In this case all that I need to do is to indicate that I want to machine all the material within the extent of the workpiece.


It will be necessary for the tool to move outside of the boundaries of the workpiece here. Also we want the plunge motions to take place outside of the material. This is set up by simply letting Pro/MANUFACTURE allow this type of motion.


There will be a number of stages in the clean up and finish machining of this type of component. An important capability is the need to machine the basic shape of the surfaces on the model ignoring for the time being, the detail features such as the boss and rib entitys on this example.
