Toggle Clamp Demo
Updated for MS6

This demo can be done as a One or Two headed demo.  It covers the following

  • Re-use of 2D drawing data
  • VSD curvature matching
  • Sketch animation
  • Assembly constraints
  • Design in the context of the assembly
  • Libraries
  • Sheet Metal
Need some pictures of final assembly

Demo Setup
 

  1. cd toggle_clamp_ms6
  2. ./install.com
  3. ideas
  4. drafting -m   from another shell and open the "Handle.asc" drawing
  5. You now should have Drafting over the top of ideas.
