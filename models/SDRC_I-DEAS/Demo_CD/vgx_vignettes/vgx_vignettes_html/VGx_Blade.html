This VGx example shows the shape and size method of creating geometry. It also shows geometry dragging with dynamic constraint recognition. Color Codeing during the constraining etc.

Begin vignette

Start with an empty screen.

Create 3 concentric circles (approx sizes)

  • 50mm
  • 85mm
  • 160mm

    Move the dimensions out and change them to arrows out.

    Anchor the circle centers.

    Add a circle tangent to the large diameter.

  • Delete the diameter dimension.
  • Drag the Center Point until the circle is tangent to the inner circle.

  • Add lines from the circle quadrant point to the center and to the last circle center.
  • Ground the first line to be Horizontal.

  • Add Angular dimension.
  • Drag to 15 degrees.

    NOTE:- As the dimension is added the free/partially/fully constrainted geometry is color coded.

    Add Arc start on the last circle and finish on the second circle.

  • Modify the radius to 75mm.
  • Add Tangency at both ends of the arc.

    Add Single line.

    Drag bottom end point of line across to get constraint at end of arc.

  • Drag the dimensions to show predicted changes.
  • Build a Stop At Intersections Section.

    Extrude 2mm Thicken

    Tech Tips

    You must be carefull about the placement of the 3 pint arc.
