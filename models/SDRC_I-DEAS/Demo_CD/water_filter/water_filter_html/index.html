This demo has been designed for the Automotive Supplier type of account who designs wthin the confines set by their customer.

This demo shows reading of an IGES file that defines the limits of the design.  The design is started using Coordinate systems and the part is created around the datum data.

The following I-DEAS features are used :-
Extrude
SIP
Align
Fillet
Shell
Reflect
Rollback
Dynamic Navigator
Geometry Drag

The demo should last about 15-20 minutes.


 
During the demo you will create the part to this point.


 
 

This shows the final part if you continued the exercise.
