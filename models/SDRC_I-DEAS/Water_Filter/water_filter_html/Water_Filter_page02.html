Initial Model File Setup

Assumptions :- Files have been unloaded onto your system and are uncompressed.

in MS5 ensure the ui.release.ms5m: 1 is set.

Setup Steps

(1) Change to the Water Filter directory.

$ cd Water_Filter

(2) Run the install program, this just copies the backup model file up from the demo_backup directory and starts ideas.

$ ./install
 

Comments shown in "green" are intended words and thoughts for the demonstrator to use.  They are NOT ment as a strict script.
