Initial Model File Setup

Assumptions :-

  • User 1 is the Part Designer.
  • User 2 is the Tool Maker
  • IMS5 is installed and the 2 machines can share data via libraries.

    Setup Steps

    (1) User 2 creates a new project.

    (2) User 2 Configures the project and adds User 2 as a member of the project.

    (3) User 2 ensures that the Email address is correct for both users.

    (4) User 2 creates a library for use during the demo.

    (5) User 2 Adopts and opens the Filler_Cap_Injection_Mold.mf* model file.

    (6) User 2 Checks-in the Filler Cap part with Keep for Reference library status.

    (7) User 2 Checks-in the Filler Cap Logo part with Do No Keep library status.

    (8) User 1 Checks-out the Filler Cap and the Filler Cap Logo parts.
